02 · How It Works

Two Retrieval Patterns

02

Memory RAG

Conversation history is prepended to each query before retrieval. Follow-up questions like "what about its performance?" resolve pronouns using prior context.

User Query + History → Augmented Query → Hybrid Retrieval → Answer
03

Branched RAG

Complex questions are decomposed into sub-questions via LLM. Each sub-question retrieves independently. Results merge, deduplicate, and rerank against the original query.

Query → Decompose → Parallel Retrieve → Merge → Rerank → Answer
